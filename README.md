#  Student Dashboard using Tableau

## Overview
The **Student Dashboard** is an interactive Tableau project designed to analyze student information using visual analytics. It provides insights into student distribution across different branches and cities through KPIs, charts, and tables. The dashboard enables users to explore the dataset with interactive filters and highlight features.

---

##  Dashboard Features

-  Total Students KPI
-  Students by Branch Analysis
-  Students by City Analysis
-  Student Details Table
-  Branch Percentage Pie Chart
-  Interactive Branch Highlight Filter

---

##  Tools & Technologies

- Tableau Public
- Microsoft Excel / CSV Dataset

---

##  Dataset Columns

- Student ID
- Student Name
- Email
- Phone Number
- City
- Branch

---

##  Dashboard Components

### 1. Total Students
Displays the total number of students in the dataset.

### 2. Students by Branch
Shows branch-wise student distribution.

### 3. Students by City
Displays the number of students from each city.

### 4. Student Details Table
Provides detailed student information including:
- City
- Email
- Branch

### 5. Branch Percentage
Pie chart representing the percentage of students in each branch.

### 6. Branch Highlight
Interactive filter that highlights a selected branch across all dashboard visualizations.
